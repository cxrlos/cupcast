# Documentation

Paper-style documentation, in reading order. Sources in `tex/` (numbered folders), built PDFs in `pdf/`. Build with `make docs` from the repo root (requires [tectonic](https://tectonic-typesetting.github.io/)).

| # | Paper | Question it answers |
|---|-------|---------------------|
| 1 | [Methodology](pdf/01-methodology.pdf) | How does the model work? Data sources, Elo ratings, the Dixon-Coles goal model, squad-quality composites, shrinkage, tournament simulation. |
| 2 | [Validation](pdf/02-validation.pdf) | Does it work? Match-level backtests on club football, tournament replays of Euro 2024 and Copa América 2024, scoring rules, calibration. |
| 3 | [Forecast](pdf/03-forecast.pdf) | What does it say? The 2026 predictions, podium probabilities, sensitivity analysis, comparison against bookmaker odds. |

Shared notation, the preamble, and the bibliography (`references.bib`) live in `tex/shared/`. Figures are generated by the pipeline into `tex/figures/`.

The working plan for the project is in [PLAN.md](PLAN.md).

The papers publish methods and findings only — the underlying datasets (API pulls, player metrics, odds snapshots) are not redistributed; the fetch scripts in `src/cupcast/fetch/` rebuild them from your own API keys.
