\begin{abstract}
I validate the forecasting model of Paper~I in two tiers. Tier~1 tests the match-level
engine where data is abundant: club football (league seasons and continental cups),
scored against bookmaker closing odds. Tier~2 replays Euro~2024 and Copa~Am\'erica~2024
end-to-end with training windows truncated to each tournament's eve, testing the full
international pipeline including tournament simulation. Probabilistic skill is measured
with strictly proper scoring rules \cite{gneitingraftery2007} --- log-loss and Brier
score --- alongside reliability diagrams; miscalibration, if found, is corrected by
isotonic regression \cite{zadroznyelkan2002} or Platt scaling \cite{platt1999} before
any 2026 forecast is issued.
\end{abstract}
